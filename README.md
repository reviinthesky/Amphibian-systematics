# Amphibian-systematics

## Определитель амфибий Тюменской области
## О проекте
Определитель амфибий — это научно-образовательный проект, разработанный для идентификации земноводных по их внешним признакам. Система позволяет пользователям быстро определить вид амфибии, основываясь на растущей базе данных характеристик.

## Авторы проекта

Главный разработчик: Олег Фалалеев (reviinthesky)

Проектный менеджер и эксперт по биологии: Снежана Ледкова

Университет: ТюмГУ, Школа естественных наук

Функционал:
Идентификация видов: определение амфибий по внешним признакам

База данных: база данных json с возможностью расширения

Интеллектуальный анализ: система сравнения характеристик

Шаблон заполнения: удобный формат для добавления новых данных

ИИ-интеграция: Рекомендации к автоматизированному заполнению с помощью нейросетей

## Особенности проекта

    Двухуровневая система поиска: разделение на хвостатых и бесхвостых амфибий

    Детальный анализ характеристик:

    Окраска тела

    Размеры

    Особенности строения

    Характерные признаки

    Гибкая система сравнения: определение степени сходства с имеющимися видами

## Технические характеристики

База данных: структурированная JSON-схема

Методы анализа: сравнительный анализ характеристик

Интерфейс: интуитивно понятный пользовательский интерфейс

Масштабируемость: возможность расширения базы данных

## Требования к системе
    Python 3.8+

    Библиотеки:

    json

    typing

    enum

    Операционная система: Windows

## Структура проекта

    data.json: основная база данных видов

    answers.json: перечень возможных характеристик

    question.py: классы для работы с данными

    identifier.py: основной код для сравнения

## Как использовать

Запустите main.exe в папке dist/

Заполните анкету с характеристиками амфибии

Получите результат идентификации

Ознакомьтесь с описанием найденного вида
