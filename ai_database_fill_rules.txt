Правила в промпт(всё что ниже, можно копировать):

Правила заполнения:
1. Для "-" использовать U+002d
2. Для "с" использовать U+0063
3. Для "C" использовать U+0043
4. Не добавляй и не стирай фигурные и квадратные скобки, которые уже стоят. Ты можешь добавлять квадратные только по надобности для выполнения правила 6;
5. После последней фигурной скобки должна стоять запятая;
6. Если в информации есть перечисление:  использовать ["data_1", "data_2"] вместо "data_1, data_2";
7. Для ключей с "has" использовать 1 - присутствует, 0 - отсутствует;
8. Если информации нет, заполнять null;
9. Если у самцов и самок есть различия помимо указанных, добавить новые поля. Если какие-то поля не используется, заполнить их null;
10. Для ключей с "count" значение должно быть типа int;
11. Для "size_cm" заполнять int типом. Например: "14-16" -> [14, 16] (то есть левая и правая граница, не в строке);
12. В "pattern_size" не должно быть чисел;
13. Для "y" использовать U+0079;
